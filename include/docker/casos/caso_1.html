Caso 01

Primeiro desafio será criar um ambiente de desenvolvimento com as seguintes ferramentas
  1. Apache
  2. PHP 7.0+
  3. MYSQL 5+
  4. PHPMyAdmin

Apache

$ sudo docker pull httpd

PHP

$sudo docker pull php
https://hub.docker.com/_/php/

MySQL

$sudo docker pull mysql
https://hub.docker.com/_/mysql/

PHPMyAdmin

$ sudo docker pull phpmyadmin/phpmyadmin
https://hub.docker.com/r/phpmyadmin/phpmyadmin/ OBS: Pasta onde ficam salvas os containes /var/lib/docker/containers/ Levantando APACHE Na pagina da documentação diz se você for rodar o httpd de um rodar isolado(Without a Dockerfile) é preciso iniciar dessa forma: docker run -dit --name apache-web -v "$PWD":/usr/local/apache2/htdocs/ httpd
