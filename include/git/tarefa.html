Referenciando issues no GITLAB

Issues no mesmo projeto:

$ git commit -m "this is my commit message. Ref #xxx"

Issues em projetos diferentes:

$ git commit -m "this is my commit message. Related to https://gitlab.com///issues/"

Para também fechar tarefa basta adicionar à mensagem do commit o seguinte:

Closes #xxx
Ex.: 
$ git commit -m "this is my commit message. Ref #38 Closes #38"
By: Guilherme Dambros
