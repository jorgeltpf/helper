COMANDOS

COMANDO DESCRIÇÃO
$ sudo docker images Listar imagens baixadas
$ sudo docker pull [nome_imagem] Baixar/pull da imagem escolhida
$ sudo docker ps Listar containers em execução
$ sudo docker ps -l Listar TODOS containers
$ sudo docker run [id_containter] Levantar/Rodar um containter
$ sudo docker stop [id_containter] Parar um containter
