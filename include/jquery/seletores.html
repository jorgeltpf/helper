Para se usar o seletor após o document ready estar pronto é preciso usar o evento ON; exemplo abaixo.

$(document).on('click', '.botaoDeletar', function(){
	alert( $(this).attr('rel') )
});
No caso do exemplo acima foi utilizado o evento click e a classe .botaoDeletar de #seletor
