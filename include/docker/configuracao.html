CONFIGURAÇÃO

Configuração global

$ GIT CONFIG --global user.name "seu_nome"
$ GIT CONFIG --global user.email "seu_email"

Criando repositório INEXISTENTE

Caso seja um repositório que você não possua... 
$ git clone git@git.emater.tche.br:49583/pbsm.git
Posteriormente seria bom , criar/editar o arquivo README.md, pois o mesmo deveria ser o primeiro contato com o projeto.

Criando repositório EXISTENTE

1º entrar na pasta do projeto 
$ git init
2º Definir a forma de comunicacao http ou ssh escolha uma delas, comigo funcionou a HTTP 
$ git remote add origin SEU_LINK_DO_REPOSITORIO(http ou ssh)
3º Escolhido a forma de comunicação basta enviar o projeto para o repositorio 
$git add .
$ git commit -m "mensagem"
$ git push -u origin master
